# Concept To 3D Pipeline UI 需求文档

## 概述

本工具是一个从概念设计到完成模型的 AIGC 辅助建模工具，UI 界面分为 **3个页面**，按流程依次推进。每个页面包含一个或多个 Pipeline，每个 Pipeline 由一系列从左到右单向运行的节点组成。

**节点基本规则：**
- 每个节点包含：一张图 或 一个/一组 3D 模型的展示区，以及若干功能按钮
- 节点只能从左向右运行
- 每个节点从前一个节点获取输入，并向下一个节点输出数据

---

## 全局 UI 规范

### 顶部导航栏
- 顶部固定导航栏，显示三个页面的标签：**Concept to Rough Model** / **Highres Model** / **Model Assemble**
- 当前页面标签高亮显示
- 页面切换时保留各页面的工作状态，不重置进度

### 整体布局
- 主内容区为横向滚动的 Pipeline 画布，节点从左到右排列
- 页面左侧或顶部可收起的**工具侧边栏**，放置辅助工具和选项
- 页面底部状态栏显示：当前运行状态、进度提示、错误信息

### 节点通用 UI 结构
每个节点由以下区域构成（从上到下）：

```
┌─────────────────────────────┐
│        节点标题              │
├─────────────────────────────┤
│                             │
│      图像 / 3D 模型展示区    │
│      （可全屏展开）          │
│                             │
├─────────────────────────────┤
│  [功能按钮组]                │
├─────────────────────────────┤
│  状态指示（待处理/运行中/完成）│
└─────────────────────────────┘
```

### 节点状态
每个节点有明确的视觉状态：

| 状态 | 视觉表现 |
| --- | --- |
| 待输入 | 灰色边框，展示区显示占位图 |
| 就绪（有输入，未运行） | 蓝色边框，展示区显示输入预览 |
| 运行中 | 蓝色边框 + 进度动画，展示区显示加载动画 |
| 完成 | 绿色边框，展示区显示输出结果 |
| 错误 | 红色边框，展示区显示错误提示 + 重试按钮 |
| 可选（Optional） | 虚线边框，可跳过 |

### 节点间连接线
- 节点之间用有向箭头连接，表示数据流向（左 → 右）
- 连接线状态随节点状态变化：灰色（未激活）/ 蓝色流动动画（传输中）/ 绿色（已完成）

### 功能按钮组设计规范
- 按钮分为两类：
  - **主操作按钮**（如"生成"、"运行"）：醒目、较大，位于按钮组右侧
  - **辅助操作按钮**（如"重置"、"选择模型"、"局部重绘"）：次要样式，位于左侧
- 运行中时主操作按钮变为"取消"，辅助按钮禁用
- 完成后主操作按钮变为"重新生成"，辅助按钮恢复可用

## 页面 1：Concept to Rough Model Pipeline

**目标：** 将概念设计稿转换为带骨骼绑定的粗模

### 布局
- 单条 Pipeline，横向排列，不支持并行
- 节点数量固定（5个），不可添加或删除
- 整体可横向滚动，节点宽度固定

### Pipeline 节点（从左到右）

| 节点 | 输入 | 输出 | 说明 |
| --- | --- | --- | --- |
| Concept | 用户上传 | 概念图 | 上传或导入概念设计稿（如 MidJourney 输出图） |
| T Pose | 概念图 | T Pose 正视图 | 生成风格一致、Pose 稳定、白平衡统一的标准正视图 |
| Multi-View | T Pose 正视图 | 多视图 | 生成风格一致、多角度稳定的多视图 |
| Rough Model | 多视图 | 粗模 | 根据多视图生成 3D 粗模 |
| Rough Model Rigging | 粗模 | 带骨骼粗模 | 对粗模进行骨骼绑定 |

### 节点 UI 细节

**Concept 节点**
- 展示区支持：拖拽上传图片 / 点击上传 / 从剪贴板粘贴
- 上传后展示图片缩略图，支持点击全屏查看
- 按钮：`上传图片` / `替换图片` / `清除`

**T Pose 节点**
- 展示区：左右对比布局，左侧显示输入概念图，右侧显示生成结果
- 支持点击结果图全屏查看
- 按钮：`生成` / `重新生成` / `全屏预览`

**Multi-View 节点**
- 展示区：网格布局展示多张视图（如 前/后/左/右）
- 每张视图可单独点击全屏查看
- 按钮：`生成` / `重新生成` / `查看全部视图`

**Rough Model 节点**
- 展示区：内嵌 3D 交互视图（支持旋转、缩放、拖拽）
- 支持切换线框模式 / 实体模式
- 按钮：`生成` / `重新生成` / `导出模型`

**Rough Model Rigging 节点**
- 展示区：内嵌 3D 交互视图，骨骼可视化叠加显示
- 按钮：`生成绑定` / `重新生成` / `导出带骨骼模型`

## 页面 2：Highres Model Pipeline

**目标：** 将粗模按部件拆分，分别制作高清模型

### 布局
- 支持**多条 Pipeline 垂直堆叠并行**，每条对应一个部件（Part）
- 每条 Pipeline 占据一行，行内节点从左到右横向排列
- 页面顶部有`+ 添加 Pipeline`按钮，每条 Pipeline 右侧有`🗑 删除`按钮
- 多条 Pipeline 可**同时独立运行**，互不影响

### Pipeline 添加 / 删除交互
- 点击`+ 添加 Pipeline`：在末尾新增一行，自动命名为 Part N（N 自动递增）
- 删除时弹出二次确认弹窗：`确认删除 Part N？此操作不可撤销`
- 支持拖拽调整 Pipeline 顺序（行与行之间拖拽）
- Pipeline 标题（Part N）支持双击重命名

### Pipeline 节点（从左到右）

| 节点 | 输入 | 输出 | 说明 |
| --- | --- | --- | --- |
| Part N Extraction | 粗模 / 概念图 | 部件图像 | 从整体中提取对应部件 |
| Part N Multi-View | 部件图像 | 部件多视图 | 生成该部件的多角度视图 |
| Part N Modify（Optional） | 部件多视图 | 修改后多视图 | 可选：对多视图进行手动修改调整 |
| Part N Rough Model 3D View | 部件多视图 | 部件粗模 | 生成该部件的粗模，提供 3D 交互视图 |
| Part N Highres Model 3D View | 部件粗模 | 部件高清模型 | 生成该部件的高清模型，提供 3D 交互视图 |
| Part N Model | 高清模型 | 最终部件模型 | 确认并输出最终部件模型 |
| Part N Re-Texturing（Optional） | 部件模型 | 重新贴图的模型 | 可选：对模型重新进行贴图 |
| Part N Region Define（Optional） | 部件模型 | 含区域定义的模型 | 可选：定义颜色调色板 Mask |

### 节点 UI 细节

**Part N Extraction 节点**
- 展示区：显示从粗模/概念图中框选出的部件区域预览
- 顶部下拉菜单：`Extraction Methods Selection`（选择提取方式，如 SAM3）
- 支持在图上手动框选/涂抹调整提取范围
- 按钮：`提取` / `重新提取` / `手动调整`

**Part N Multi-View 节点**
- 展示区：网格布局展示多视图
- 按钮：`生成` / `重新生成` / `查看全部视图`

**Part N Modify 节点（Optional）**
- 展示区：图像编辑界面，支持画笔涂抹、局部重绘
- 节点默认收起（虚线边框），点击`展开`后激活
- 按钮：`局部重绘` / `撤销` / `重置` / `确认修改`

**Part N Rough Model 3D View 节点**
- 展示区：内嵌 3D 交互视图（旋转、缩放、拖拽）
- 支持线框/实体切换
- 按钮：`生成` / `重新生成` / `导出粗模`

**Part N Highres Model 3D View 节点**
- 展示区：内嵌 3D 交互视图
- 支持与左侧粗模**左右分屏对比**
- 按钮：`生成` / `重新生成` / `对比粗模` / `导出高清模型`

**Part N Model 节点**
- 展示区：最终部件模型 3D 展示
- 用户在此节点做最终确认，确认后模型进入 Model Gallery
- 按钮：`确认` / `返回修改` / `导出`

**Part N Re-Texturing 节点（Optional）**
- 节点默认收起，需用户主动展开
- 按钮：`重新贴图` / `选择贴图` / `重置` / `确认`

**Part N Region Define 节点（Optional）**
- 节点默认收起，需用户主动展开
- 展示区：模型上叠加颜色 Mask 编辑界面
- 按钮：`添加区域` / `删除区域` / `调色板编辑` / `确认`

### 辅助元素
- **Extraction Methods Selection**：每条 Pipeline 的 Extraction 节点内集成，下拉选择提取方式

## 页面 3：Model Assemble Page & Transfer Rigging Page

**目标：** 将各部件高清模型对照粗模进行拼装，完成最终完整模型

### 布局
- 页面主体为大型 3D 视口，占据中央大部分空间
- 左侧面板：Outliner（层级管理）+ Mesh Gallery / Model Gallery / Pose Gallery 标签切换
- 右侧面板：Landmark Points 工具 + Align Tools + 属性编辑
- 底部：Transfer Rigging Page 入口按钮

### 主要功能区

| 功能区 | 位置 | 说明 |
| --- | --- | --- |
| Model Gallery | 左侧面板 | 高清部件模型素材库，拖拽或双击加入场景 |
| Mesh Gallery | 左侧面板 | 网格素材库，拖拽或双击加入场景 |
| Pose Gallery | 左侧面板 | 姿态库，点击应用到当前模型 |
| Mesh / Outliner | 左侧面板顶部 | 场景层级树，显示所有已加入部件，支持显示/隐藏/锁定 |
| Landmark Points | 右侧面板 | 在模型上点击添加标记点，支持增删改查 |
| Align Tools | 右侧面板 | 基于 Landmark Points 执行对齐，显示对齐误差 |
| Transfer Rigging Page | 底部入口 | 跳转至骨骼迁移页面 |

### Mesh Gallery UI 细节
- 展示方式：网格缩略图列表，支持搜索和标签筛选
- 每个 Mesh 卡片显示：缩略图 + 名称 + 面数信息
- 操作方式：拖拽到 3D 视口 / 双击加入场景
- 支持从外部导入新 Mesh（`+ 导入`按钮）

### 对齐流程（已开发）
1. 在部件模型上打 Landmark Points
2. 使用最小二乘法计算最优刚体变换（含缩放）
3. 通过 Align Tools 执行自动对齐
4. 手动微调（如需）

---

## 整体数据流向

```
[页面1] 概念稿 → T Pose → 多视图 → 粗模 → 带骨骼粗模
                                          ↓
[页面2] 部件拆分 → 部件多视图 → （修改）→ 部件粗模 → 部件高清模型 → （重贴图/区域定义）→ Model Gallery
                                                                                        ↓
[页面3] Model Gallery → 3D视口拼装（Landmark对齐）→ Transfer Rigging → 完整高清模型
```
